إعداد Firebase Firestore

ربط نظام الصقري للاتصالات مع قاعدة بيانات Firebase

1

إنشاء مشروع Firebase

ملاحظة: ستحتاج إلى حساب Google لإنشاء مشروع Firebase
  1. اذهب إلى Firebase Console
  2. اضغط على "إضافة مشروع" (Add Project)
  3. أدخل اسم المشروع: alsaqri-phones
  4. اختر "تمكين Google Analytics" (اختياري)
  5. اضغط "إنشاء المشروع"
2

إضافة تطبيق ويب

  1. في لوحة تحكم المشروع، اضغط على أيقونة الويب
  2. أدخل اسم التطبيق: alsaqri-phones-web
  3. اختر "إعداد Firebase Hosting" (اختياري)
  4. اضغط "تسجيل التطبيق"
3

الحصول على إعدادات Firebase

انسخ إعدادات Firebase من لوحة التحكم:

const firebaseConfig = { apiKey: "your-api-key-here", authDomain: "your-project-id.firebaseapp.com", projectId: "your-project-id", storageBucket: "your-project-id.appspot.com", messagingSenderId: "your-sender-id", appId: "your-app-id" };

تفعيل Firestore Database

مهم: يجب تفعيل Firestore Database لحفظ البيانات
  1. في لوحة تحكم Firebase، اذهب إلى "Firestore Database"
  2. اضغط "إنشاء قاعدة بيانات"
  3. اختر "بدء في وضع الإنتاج"
  4. اختر موقع قاعدة البيانات (الأقرب للسعودية)
  5. اضغط "تم"
6

إعداد قواعد الأمان

في Firestore Database، اذهب إلى "Rules" واكتب القواعد التالية:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // قواعد للهواتف match /phones/{phoneId} { allow read, write: if true; } // قواعد للأكسسوارات match /accessories/{accessoryId} { allow read, write: if true; } // قواعد للمبيعات match /sales/{saleId} { allow read, write: if true; } // قواعد لأنواع الهواتف match /phone_types/{typeId} { allow read, write: if true; } // قواعد لفئات الأكسسوارات match /accessory_categories/{categoryId} { allow read, write: if true; } } }
تحذير: هذه القواعد تسمح بالوصول للجميع. للاستخدام الإنتاجي، يجب إضافة المصادقة.
